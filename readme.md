## About Project
The Symfony application - a class for validating an arbitrary set of data from a form.

## Requirements
  * PHP 7.2.5 or higher;
  * and the [usual Symfony application requirements][1].

## Quick Start
```shell
$ git clone https://github.com/vol-mir/valid-form.git
$ cd valid-form
$ composer install
```
